    Function `collect_team_rosters(client: NBAClient, conn: sqlite3.Connection, seasons: list[str] = None) -> dict`:
    - Default seasons to SEASONS if None
    - Get all teams: `teams = client.get_all_teams()` — returns list of dicts with keys: id, full_name, abbreviation, nickname, city, state, year_founded
    - First, seed the teams table: for each team call `queries.upsert_team(conn, team["id"], team["abbreviation"], team["full_name"])`
    - Build work list: all (team_id, season) pairs = 30 teams × 5 seasons = 150 items
    - For each (team_id, season), check if already completed: query collection_progress where entity_type='team_roster', entity_id=team_id, season=season, status='completed'. If yes, skip.
    - For remaining items, iterate with `tqdm(remaining, desc="Collecting rosters")`:
      - Try: call `df = client.fetch_team_roster(team_id, season)`
      - Parse roster DataFrame: expected columns include PLAYER_ID, PLAYER (name), POSITION, NUM, SEASON
      - For each row in df: call `queries.insert_team_roster(conn, team_id, int(row["PLAYER_ID"]), season)`
      - Also upsert player: `queries.upsert_player(conn, int(row["PLAYER_ID"]), row["PLAYER"], True, row.get("POSITION"), team_id)`
      - Call `queries.mark_progress(conn, 'team_roster', team_id, season, 'completed')`
      - Except Exception as e: call `queries.mark_progress(conn, 'team_roster', team_id, season, 'failed', str(e))`, log error, continue
    - Return {"total": len(teams) * len(seasons), "completed": completed_count, "failed": failed_count, "skipped": skipped_count}

    Function `_parse_minutes(min_str) -> float`:
    - NBA API returns MIN as "MM:SS" string or sometimes as float/int
    - If input is None or empty: return 0.0
    - If input is already numeric (int/float): return float(input)
    - If input is string containing ":": split on ":", return int(parts[0]) + int(parts[1]) / 60.0
    - Otherwise: try float(min_str), return 0.0 on failure

    Function `collect_player_gamelogs(client: NBAClient, conn: sqlite3.Connection, seasons: list[str] = None) -> dict`:
    - Default seasons to SEASONS if None (per D-01: 5 seasons 2020-2025)
    - Get all active players from players table: `cursor = conn.execute("SELECT DISTINCT player_id, full_name FROM players WHERE is_active = 1"); players = cursor.fetchall()`
    - If players table is empty, fall back to client.get_all_active_players() and seed: for each player call queries.upsert_player() (per D-02: all 450+ active players)
    - Use per-season calls (NOT SeasonAll.all) per research recommendation — smaller responses, better progress tracking per season
    - Build full work list: all (player_id, season) pairs for each active player × each season in SEASONS
    - Seed collection_progress: for each (player_id, season) pair, insert with status='pending' if not already present: `INSERT OR IGNORE INTO collection_progress (entity_type, entity_id, season, status) VALUES ('player_gamelog', ?, ?, 'pending')`
    - Get remaining work: query collection_progress WHERE entity_type='player_gamelog' AND status != 'completed'
    - Initialize counters: games_inserted = 0, players_completed = 0, failed = 0, skipped = 0
    - Iterate with `tqdm(remaining_work, desc="Collecting game logs")`:
      - Extract player_id, season from work item
      - Try:
        - Call `df = client.fetch_player_gamelog(player_id, season)`
      - Except ValueError (empty response — player may not have played that season):
        - logger.info(f"No games for player {player_id} in {season}")
        - queries.mark_progress(conn, 'player_gamelog', player_id, season, 'completed')
        - skipped += 1; continue
      - Except Exception as e:
        - logger.error(f"Failed for player {player_id} season {season}: {e}")
        - queries.mark_progress(conn, 'player_gamelog', player_id, season, 'failed', str(e))
        - failed += 1; continue
      - If df is not empty:
        - Rename columns using GAMELOG_COLUMN_MAP (only rename columns that exist in df)
        - If 'player_id' not in df.columns after rename: add df["player_id"] = player_id
        - Set df["season"] = season
        - Parse minutes: df["min"] = df["min"].apply(_parse_minutes)
        - Add DNP flag: df["is_dnp"] = 0 (all real game logs, not synthesized)
        - Select only the columns matching the player_game_logs schema: ["player_id", "game_id", "season", "game_date", "matchup", "wl", "min", "pts", "reb", "ast", "stl", "blk", "fg3m", "fgm", "fga", "ftm", "fta", "oreb", "dreb", "tov", "pf", "plus_minus", "is_dnp"]
        - Fill NaN values with 0 for numeric columns (pts, reb, etc.)
        - Call queries.insert_game_logs(conn, df)
        - games_inserted += len(df)
      - queries.mark_progress(conn, 'player_gamelog', player_id, season, 'completed')
      - players_completed += 1
    - Return {"players_completed": players_completed, "games_inserted": games_inserted, "failed": failed, "skipped": skipped, "remaining": len(remaining_work) - players_completed - failed - skipped}
