    **server/pipeline/nba_client.py** — Create with:

    Module-level `setup_cached_session(cache_path: str)` function:
    ```python
    from requests_cache import CachedSession
    from nba_api.stats.library.http import NBAHTTP

    def setup_cached_session(cache_path: str) -> CachedSession:
        session = CachedSession(
            cache_name=cache_path,
            backend="sqlite",
            expire_after=None,
            allowable_methods=["GET", "POST"],
            stale_if_error=True,
        )
        session.headers.update({
            "User-Agent": "Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/120.0.0.0 Safari/537.36",
            "Referer": "https://www.nba.com/",
            "Accept-Language": "en-US,en;q=0.9",
        })
        NBAHTTP.set_session(session)
        return session
    ```

    Class `NBAClient`:
    - Class constant `MIN_DELAY = 0.6` (600ms between calls — per DATA-03 and research)
    - `__init__(self, cache_path: str = None)`:
      - Default cache_path to CACHE_PATH from server.pipeline
      - `self._last_call = 0.0`
      - `self._session = setup_cached_session(cache_path)`
      - `self.logger = logging.getLogger(__name__)`

    - `_enforce_rate_limit(self)`:
      ```python
      elapsed = time.time() - self._last_call
      if elapsed < self.MIN_DELAY:
          time.sleep(self.MIN_DELAY - elapsed)
      self._last_call = time.time()
      ```

    - Every fetch method gets this tenacity decorator:
      ```python
      @retry(
          stop=stop_after_attempt(5),
          wait=wait_exponential_jitter(initial=0.6, max=30, jitter=2),
          retry=retry_if_exception_type((ConnectionError, TimeoutError, ValueError)),
          before_sleep=before_sleep_log(logger, logging.WARNING),
      )
      ```
      Note: `logger` must be module-level for before_sleep_log. Retry on ValueError too — raised when NBA API returns empty DataFrame.

    - `fetch_player_gamelog(self, player_id: int, season: str) -> pd.DataFrame`:
      - Call self._enforce_rate_limit()
      - `from nba_api.stats.endpoints import playergamelog`
      - `df = playergamelog.PlayerGameLog(player_id=player_id, season=season, season_type_all_star="Regular Season", timeout=30).get_data_frames()[0]`
      - If df is empty: raise ValueError(f"Empty gamelog for player {player_id} season {season}")
      - self.logger.info(f"Fetched {len(df)} games for player {player_id} season {season}")
      - Return df

    - `fetch_team_roster(self, team_id: int, season: str) -> pd.DataFrame`:
      - Call self._enforce_rate_limit()
      - `from nba_api.stats.endpoints import commonteamroster`
      - `df = commonteamroster.CommonTeamRoster(team_id=team_id, season=season, timeout=30).get_data_frames()[0]`
      - If df is empty: raise ValueError(f"Empty roster for team {team_id} season {season}")
      - Return df

    - `fetch_team_schedule(self, team_id: int, season: str) -> pd.DataFrame`:
      - Call self._enforce_rate_limit()
      - `from nba_api.stats.endpoints import leaguegamefinder`
      - `df = leaguegamefinder.LeagueGameFinder(team_id_nullable=team_id, season_nullable=season, player_or_team_abbreviation="T", season_type_nullable="Regular Season", timeout=30).get_data_frames()[0]`
      - If df is empty: raise ValueError(f"Empty schedule for team {team_id} season {season}")
      - Return df

    - `fetch_team_advanced_stats(self, season: str) -> pd.DataFrame`:
      - Call self._enforce_rate_limit()
      - `from nba_api.stats.endpoints import leaguedashteamstats`
      - `df = leaguedashteamstats.LeagueDashTeamStats(measure_type_detailed_defense="Advanced", season=season, season_type_all_star="Regular Season", per_mode_detailed="PerGame", timeout=30).get_data_frames()[0]`
      - If df is empty: raise ValueError(f"Empty team stats for season {season}")
      - Verify "DEF_RATING" in df.columns — if not, log warning: f"Expected DEF_RATING column not found. Available columns: {df.columns.tolist()}"
      - Return df
