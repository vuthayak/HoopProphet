  <action>
    Create server/pipeline/processors/rolling_features.py implementing `compute_rolling_features(played_df: pd.DataFrame) -> pd.DataFrame`.

    **Function signature and contract:**
    - Input: DataFrame of game logs already filtered to `is_dnp == 0` (caller handles DNP filtering per D-05). Expected columns: player_id, game_id, season, game_date, matchup, and all 16 stat columns from STAT_COLS.
    - Output: Same DataFrame with new rolling/expanding feature columns appended. Original columns preserved.

    **Implementation steps:**

    1. Import from feature_config: `STAT_COLS, PRIMARY_STATS, SECONDARY_STATS, COMBO_STATS, WINDOWS_PRIMARY, WINDOWS_SECONDARY`

    2. Sort: `df = played_df.sort_values(['player_id', 'game_date']).copy()`

    3. Compute combo stat raw columns (per D-02 — PRA, PA, PR):
       ```python
       for combo_name, component_stats in COMBO_STATS.items():
           df[combo_name] = df[component_stats].sum(axis=1)
       ```
       This creates columns: pra = pts+reb+ast, pa = pts+ast, pr = pts+reb

    4. Compute rolling averages and standard deviations for all 16 base stats:
       For each stat in STAT_COLS:
       - Determine windows: `WINDOWS_PRIMARY` if stat in PRIMARY_STATS else `WINDOWS_SECONDARY`
       - For each window w:
         - min_periods: 1 for w in [5, 10], 10 for w == 20 (per research recommendation)
         - Rolling mean: `df.groupby('player_id')[stat].rolling(w, min_periods=mp).mean().reset_index(level=0, drop=True)`
           → column name: `{stat}_avg_L{w}`
         - Rolling std: same with `.std()`
           → column name: `{stat}_std_L{w}`

    5. Compute rolling averages and standard deviations for combo stats (pra, pa, pr):
       For each combo_name in COMBO_STATS:
       - Use WINDOWS_PRIMARY [5, 10, 20]
       - Same rolling mean/std pattern as step 4
       - Column names: `{combo}_avg_L{w}`, `{combo}_std_L{w}`

    6. Compute season-to-date expanding features (per research recommendation):
       Group by ['player_id', 'season']:
       - For each stat in PRIMARY_STATS:
         - Expanding mean: `df.groupby(['player_id', 'season'])[stat].expanding(min_periods=1).mean().reset_index(level=[0,1], drop=True)`
           → column name: `{stat}_season_avg`
       - Games played count: `df.groupby(['player_id', 'season']).cumcount() + 1`
         → column name: `games_played_season`

    7. **CRITICAL — Apply .shift(1) temporal guard (FEAT-09):**
       Collect ALL computed feature column names:
       ```python
       rolling_cols = [c for c in df.columns
                       if '_avg_L' in c or '_std_L' in c
                       or '_season_avg' in c or c == 'games_played_season']
       df[rolling_cols] = df.groupby('player_id')[rolling_cols].shift(1)
       ```
       This ensures game N's features contain ONLY data through game N-1.

    8. Return df with all new columns.

    **Column count estimate:** 16 stats × (2-3 windows) × 2 (avg+std) + 3 combos × 3 windows × 2 + 8 season_avg + 1 games_played ≈ 100+ new columns.

    Use `logging.getLogger(__name__)` for logging. Log start/end and column count.
