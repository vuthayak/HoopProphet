  <action>
    Create 5 custom hooks following the pattern from RESEARCH.md Pattern 3 (useState + useEffect + fetch with cancellation). All hooks use api client from Plan 01.

    1. `useSearch(query)` — Per D-15 (persistent search) and UI-SPEC Interaction Pattern 1:
       - Debounces input by 300ms (DEBOUNCE_MS)
       - Only calls API when query length >= SEARCH_MIN_CHARS (2)
       - Returns { results, loading, error }
       - Cancels stale requests via AbortController
       - On selection, calls `onSelect(player)` callback with player object
       - Empties results when query is cleared

    2. `usePlayerData(playerId)` — Per D-16 (parallel fetch on page load):
       - Fetches player detail, props, and lines in parallel (Promise.all)
       - Returns { player, props, lines, alerts, loading, error }
       - Cancels stale requests when playerId changes
       - Shows toast on error via react-hot-toast

    3. `useGameLogs(playerId, limit=50)`:
       - Fetches /api/players/{id}/gamelogs?limit={limit}
       - Returns { gamelogs, loading, error }
       - Cancels on playerId change

    4. `useNews(playerId)`:
       - Fetches /api/players/{id}/news
       - Returns { news, alerts, staleWarning, loading, error }
       - Extracts alerts array and stale_warning from response

    5. `useHitRates(playerId, stat, line)` — Per D-08 (adjustable lines) and Pitfall 7 (debounce slider):
       - Fetches /api/players/{id}/hitrates?stat={stat}&line={line}
       - 300ms debounce on stat/line changes (prevents rapid refetch on slider drag)
       - Returns { hitRates, loading, error }
       - Use useEffect with setTimeout for debounce, clear on cleanup

    All hooks follow this pattern:
    ```js
    import { useState, useEffect } from 'react';
    import { api } from '../api/client';

    export function useHookName(/* params */) {
      const [data, setData] = useState(null);
      const [loading, setLoading] = useState(true);
      const [error, setError] = useState(null);

      useEffect(() => {
        let cancelled = false;
        // ... fetch logic with api.XXX
        return () => { cancelled = true; };
      }, [/* deps */]);

      return { data, loading, error };
    }
    ```

    Create `hoopprophet/src/__tests__/hooks.test.js` testing:
    - useSearch: returns empty results for short queries, sets loading on fetch
    - usePlayerData: returns null before fetch completes, handles errors
    - useHitRates: debounce delays fetch call
    Use vi.mock to mock api module, vi.useFakeTimers for debounce testing.
