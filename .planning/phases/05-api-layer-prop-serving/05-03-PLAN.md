## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| client → API | Untrusted HTTP input crossing network. All query params and path params are untrusted. |
| API → services | Internal boundary. Services receive validated IDs and params from routers. |
| services → SQLite | Trusted local file. Queries use parameterized inputs from service functions. |
| services → model artifact | Trusted local file. Read-only access to joblib artifact. |
| services → features.parquet | Trusted local file. Read-only access for prediction feature lookup. |

| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-05-08 | Elevation of Privilege | Model artifact path injection | mitigate | MODEL_ARTIFACT_PATH is from config, not user input; no path traversal possible |
| T-05-09 | Information Disclosure | Stack traces in error responses | mitigate | FastAPI's default exception handler hides stack traces in production; use proper exception classes |
