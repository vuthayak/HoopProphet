         e) Concatenate all per-prediction rows into a single DataFrame
         f) Return BacktestResult with predictions_df, fold_summaries, n_folds, seasons
       - IMPORTANT: The fold_summaries list stores per-fold metrics dicts for downstream reporting (Plan 02 uses these, Plan 03 outputs them)

       - Ensure stat_type is extracted from val_df as integer (STAT_TYPE_MAP values) and stored as-is; stat_type_name mapping happens in metrics (Plan 02)
